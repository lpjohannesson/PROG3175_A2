# Greetings API

## Endpoints

* ```/Greet```
    * timeOfDay
    * language
    * tone
* ```/GetAllTimesOfDay```
* ```/GetSupportedLanguages```

## Example

Request: ```http://localhost/Greet?timeOfDay=Morning&language=English&tone=Formal```

Response: ```{"greetingMessage":"Good morning"}```
